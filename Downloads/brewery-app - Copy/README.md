# MvndiCraft App

All-in-one companion app for the MvndiCraft medieval Minecraft server.

**Modules:**
- ⚗ Brewery — Recipe tracker, barrel aging timers, brew log
- ⚔ Siege Planner — Plan attacks/defenses with countdowns and role assignments
- 🏰 Nation Dashboard — Towns, wars, allies, member roster
- ⚒ Smithy — Blacksmith order tracker with material inventory
- 🚢 Fleet — Ship registry with condition tracking and voyage log
- ⚡ Skills — RPG skill tree planner with saved builds
- 💰 Trade Ledger — Log trades and track debts
- 📜 War Chronicle — Battle history with win/loss statistics

---

## Running locally
```bash
npm install
npm start
```

## Build installer
```bash
npm install
npm run build
```

Output: `dist/MvndiCraft-Setup.exe`

**IP:** mc.mvndicraft.net
